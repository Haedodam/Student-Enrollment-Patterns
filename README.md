# Student Enrollment Patterns in a Public Affairs Graduate Program: A 10-Year Trend Analysis

## Overview
This repository contains the analysis code and reports for a capstone project examining student enrollment patterns across UIC's MPA, MPP, and MSCA programs from Fall 2015 through Spring 2025.

## Contents
- `PA590_Quarto_Report.qmd` — Reproducible analysis report (Quarto)
- `PA590_Quarto-Report.pdf` — Rendered PDF version of the Quarto report
- `02_Report_PPMA.pdf` — Final project report

## Research Questions
1. Are students following the recommended course sequence?
2. Are there signs of course overload?
3. How has time to degree changed over time?
4. How have enrollment patterns across concentrations shifted?
5. How much do students deviate from the recommended course plan?
6. How has participation in nonprofit coursework changed?
7. To what extent do students retake courses?
